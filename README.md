# AI Red Teaming

A collection of research projects exploring offensive AI security techniques, including jailbreaking, autonomous exploitation, and prompt injection.

## Projects

| Project | Description |
|---|---|
| [local-echo-chamber](./local-echo-chamber) | Evaluates the [Echo Chamber](https://neuraltrust.ai/blog/echo-chamber-context-poisoning-jailbreak) context-poisoning jailbreak on Small Language Models |
| [cyber-agent](./cyber-agent) | RL agent trained to autonomously exploit networks under a post-breach assumption |
| [indirect-prompt-injection](./indirect-prompt-injection) | Tests indirect prompt injection attacks in a RAG pipeline |
